# FANZA同人 サークルBlacklist Chrome拡張機能

FANZA同人（DMM同人）で、特定のサークル（出品者）やジャンルの商品を一覧・検索・ランキングから非表示にする Chrome 拡張機能です。
趣味に合わない・興味のないサークルや、質の悪い出品を繰り返すサークルの作品をサークル単位でリストから除外できます。
要素を非表示にするだけなのでページを超えて表示は繰り上がりません。単純に表示件数が減ります。

## 主な機能

- **サークル単位の非表示**
  各商品のサークル名の横に表示される「🚫 非表示」ボタンをクリックすると、そのサークルの商品が一覧から消えます。サークルはIDで識別されます。
- **カテゴリ単位の表示切り替え**
  コミック・CG・動画・ボイス・ボイコミ・ゲームなどのカテゴリごとに表示／非表示を切り替えられます。初期状態はすべて表示で、非表示にしたいカテゴリだけを設定します。閲覧したページから新しいカテゴリを自動検出して設定に追加します。
- **設定ページ**
  非表示サークルの一覧表示・名前の編集・解除、カテゴリの表示切り替え、設定のエクスポート／インポートができます。
- **設定の永続化**
  設定は `chrome.storage.local` に保存され、ブラウザやPCを再起動しても保持されます（拡張機能を削除すると消えます）。

## 対応ページ

| ページ | URL 例 | カード単位 |
|---|---|---|
| 作品一覧・セール等 | `/dc/doujin/-/list/=/...` | `li.productList__item` |
| キーワード検索（同人内） | 同上（同一レイアウト） | `li.productList__item` |
| ランキング | `/dc/doujin/-/ranking-all/=/...` | `li.rank-rankListItem` |

> 対象は FANZA**同人**（`www.dmm.co.jp/dc/doujin/`）のページです。全サービス横断検索（`www.dmm.co.jp/search/`）は対象外です。

## インストール（パッケージ化されていない拡張機能）

1. Chrome で `chrome://extensions` を開く
2. 右上の「**デベロッパーモード**」をONにする
3. 「**パッケージ化されていない拡張機能を読み込む**」をクリック
4. `manifest.json` があるフォルダを選択する

コードを変更したら、`chrome://extensions` の🔄（更新）ボタンで再読み込みします。

## 使い方

### サークルを非表示にする
FANZA同人の一覧・ランキングページで、各商品のサークル名の下にある「🚫 非表示」ボタンをクリックします。以降、そのサークルの商品は自動的に非表示になります。

### カテゴリを非表示にする
設定ページの「カテゴリの表示切り替え」で、カテゴリごとにチェックを外すと非表示になります（チェック＝表示）。**初期状態はすべて表示**です。閲覧したページで見つかったカテゴリは自動的に一覧へ追加されます。

### 設定ページを開く
- ツールバーの拡張機能アイコン → ポップアップの「⚙ 詳細設定」、または
- `chrome://extensions` → 本拡張機能の「詳細」→「拡張機能のオプション」

設定ページでは以下ができます。
- 非表示サークルの一覧・名前編集・個別解除・全解除・手動追加（maker ID 指定）
- カテゴリの表示切り替え（チェックを外すと非表示。閲覧したページから自動追加）
- **エクスポート**: 設定を JSON ファイルに保存
- **インポート**: JSON ファイルから設定を追加（既存設定にマージ）

## maker ID について

サークルは maker ID（数字）で識別します。商品ページ内のサークルリンクの URL に含まれます。

```
https://www.dmm.co.jp/dc/doujin/-/list/=/article=maker/id=218880/
                                                        ^^^^^^  ← maker ID
```

設定ページからは、この ID を直接指定してサークルを追加することもできます。

## データの保存形式

`chrome.storage.local` に以下のキーで保存されます。

- `blockedCircles` … 非表示サークル。`{ "makerID": "サークル名", ... }`
- `blockedGenres` … 非表示にするカテゴリ名の配列。初期値は `[]`（すべて表示）
- `seenGenres` … これまで閲覧して見つかったカテゴリ名の配列（設定画面の候補生成に使用）

エクスポート／インポートの対象は `blockedCircles` と `blockedGenres` です。
